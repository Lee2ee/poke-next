# Poke-Next 프로젝트 실행 방법

## 설치 및 실행

아래 명령어를 순서대로 실행하여 프로젝트를 로컬에서 실행할 수 있습니다.

```sh
npm install
npm run dev
```

## 접속 방법

브라우저에서 다음 주소로 접속하세요.

```
localhost:3000/v3
```
